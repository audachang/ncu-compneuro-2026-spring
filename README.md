## Course Description
This course bridges classical psychological theory with modern AI and Human-Computer Interaction (HCI). It empowers students to Create (experiments), Decode (neural signals), Predict (behavior), and Reproduce (open science).

## Syllabus
- Week 01: Introduction and Environment Setup
- Week 02: Python Foundations for Psychology
- Week 03: The Digital Lab - PsychoPy Basics
- Week 04: Experimental Design and Data Logging
- Week 05: Data Analysis with Pandas
- Week 06: Neural Feature Engineering - ROI Extraction
- Week 07: Biological Signal Processing
- Week 08: Midterm Project
- Week 09: MVPA and Neural Decoding
- Week 10: Machine Learning Classification
- Week 11: Behavioral Prediction Models
- Week 12: Feature Importance and Model Evaluation
- Week 13: Industry Applications - UX and Clustering
- Week 14: Reproducibility and Docker
- Week 15: Final Project Workshop
- Week 16: Final Project Showcase
